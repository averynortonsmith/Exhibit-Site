Exhibit 3.0 Syntax Notes:

Exhibit 3.0 Syntax Notes:

Exhibit 3.0 is similar syntactically to Exhibit 2.0, with one key difference: in Exhibit 2.0, the element attributes used to specify settings for Exhibit elements were prefixed with "ex:" In Exhibit 3.0, this prefix has been replaced with the more standard "data-ex-" In addition, the setting names which follow this prefix, which were before written in "camelCase", are now written in "hyphen-case". For example:

"ex:viewClass" becomes "data-ex-view-class"
"ex:content" becomes "data-ex-content"
"ex:colorKey" becomes "data-ex-color-key"

With these differences in mind, much of the documentation for Exhibit 2.0 is still a relevant and useful reference for Exhibit 3.0.
